MY JOURNEY IN WEB PROGRAMMING

A journey through my academic progress

INTRODUCTION TO HTML

During my prelims, I learned about the fundamental concepts of database systems, starting with data models that provide structured representations of real-world data through entities, attributes, and relationships.

I learned the basics of web development using HTML and CSS. HTML taught me how to structure web content using tags for text, links, images, and forms. I discovered how to organize information with lists, tables, and proper document layout. CSS showed me how to style these elements by changing colors, fonts, spacing, and backgrounds. I practiced connecting CSS to HTML through internal and external styling methods. The lessons emphasized how HTML defines content while CSS controls its appearance, working together to create functional and visually appealing web pages. These fundamentals prepared me for more advanced web development topics in the coming terms.

This topic was crucial for setting the stage for my learning journey. It helped me identify my strengths and areas needing improvement, which I could then focus on during subsequent terms.

INTRODUCTION TO CSS

I learned responsive web design using CSS Grid and media queries to create layouts that adapt to different screen sizes. I also studied JavaScript basics, including variables, operators, loops, and conditional statements to add interactivity to websites. The module covered how to make images and videos responsive and introduced JavaScript functions for dynamic web content.

This topic marked a turning point where theoretical knowledge began transforming into practical skills. The projects during this period helped me develop critical thinking and analytical abilities.

INTRODUCTION TO JAVA SCRIPT

In this topic I've learned about JavaScript functions and error handling. Functions help organize code into reusable blocks, using parameters to pass data and return to output results. I practiced declaring functions, calling them, and using local/global variables. JavaScript functions for reusable code and error handling using try-catch. The module covered common errors (SyntaxError, TypeError) and debugging techniques like console.log() and breakpoints to troubleshoot code effectively.

It was particularly valuable for developing my technical skills. Learning to work with different platforms and analytics tools gave me practical experience.

INTRODUCTION TO ASP.NET

ASP.NET brought everything together Web Programming. This required integrating all the knowledge and skills gained throughout the semester into a cohesive strategy.

In this term, I was introduced to ASP.NET, Microsoft's framework for building dynamic web applications. The module covered .NET platform fundamentals, ASP.NET architecture (Web Forms, MVC, Web Pages), and ASP.NET Core's cross-platform capabilities. I learned about key components like Program.cs configuration, middleware, dependency injection, and error handling techniques. The course highlighted ASP.NET's advantages including Web APIs, real-time communication with SignalR, and cloud-ready application development using C# programming language.

Looking back on the entire topics, I can see growth in both my knowledge and skills. The progressive structure of the course allowed me to build competence step by step, culminating in the ability to create and execute complete strategies.
